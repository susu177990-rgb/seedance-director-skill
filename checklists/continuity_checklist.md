# Continuity Checklist

## Track Before Writing

For every subject:

```text
Who/what:
Initial position:
Initial body/object state:
Movement path:
Final position:
Visible anchors:
```

## Track Across Shots

- frame left/right consistency
- foreground/background relationship
- hand/object ownership
- sitting/standing/leaning state
- gaze direction
- emotion progression
- light/weather state
- wardrobe/hair state

## If Continuity Breaks

Use one of these solutions:

- Add a transition movement.
- Keep the subject in the same position.
- Reframe without moving the subject.
- Mention the same spatial anchor.
- Add blurred continuity evidence.
- Shorten or simplify the action.
