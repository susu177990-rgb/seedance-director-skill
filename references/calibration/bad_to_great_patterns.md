# Bad To Great Director Patterns

Use this file as a quick rewrite reference when a prompt sounds complete but ordinary.

## Pattern 1: Generic Mood To Attention Path

Weak:

```text
画面很孤独，电影感强。
```

Rewrite direction:

```text
第一眼让观众看空椅子，第二眼看角色没有坐下的手，最后让视线停在椅子上还没有消失的身体压痕。
```

Rule:

- Mood is not a look. Mood is attention order plus what the frame refuses to resolve.

## Pattern 2: Product Display To Product Reveal Mechanism

Weak:

```text
产品放在桌上，高级精致。
```

Rewrite direction:

```text
产品不直接居中展示，先让一条窄光扫到瓶肩，手移开前景包装纸后，瓶身边缘才完整进入焦点，最后停在材质高光上。
```

Rule:

- A product should be revealed by light, hand, material, reflection, packaging, scale, or use.

## Pattern 3: Emotional Label To Performance Leakage

Weak:

```text
她表面平静但很难过。
```

Rewrite direction:

```text
她没有看手机，右手继续搅动杯子但速度越来越慢，嘴角维持礼貌弧度，下颌却先绷住，呼吸停半拍后才放下勺子。
```

Rule:

- Good performance separates mask and leakage.

## Pattern 4: Constant Motion To Purposeful Pause

Weak:

```text
镜头一直推近，营造高级电影感。
```

Rewrite direction:

```text
镜头只在产品还被遮挡时缓慢靠近；产品进入清晰焦点后停止移动，停住2秒让材质和品牌气质落下来。
```

Rule:

- Movement should reveal. Pause should make the reveal matter.

## Pattern 5: Decorated Space To Staged Relationship

Weak:

```text
房间里有窗、桌子、植物、杯子，氛围温暖。
```

Rewrite direction:

```text
窗光从左后方落到杯沿，桌面反光把光带引向角色停住的手，植物只作为右侧前景遮挡，让画面像从真实空间里偷看到这个动作。
```

Rule:

- Space elements must guide, block, pressure, reveal, or echo the subject.

## Pattern 6: Over-Polished Casual Footage To Believable Observation

Weak:

```text
随手拍也要高级，镜头稳定推近，食物很精致。
```

Rewrite direction:

```text
保留轻微手持晃动，食物递近时镜头自然后退半步，焦点短暂落到热气，再重新对上朋友的手，像真实手机反应。
```

Rule:

- Casual excellence comes from controlled authenticity, not commercial polish.

## Pattern 7: Action Chaos To Readable Pressure

Weak:

```text
快速奔跑、镜头猛烈晃动、动作紧张。
```

Rewrite direction:

```text
先用脚步声让角色肩膀僵住，再让墙上影子进入画面，最后角色退入右侧阴影。镜头保持中景，空间清楚，紧张来自他不敢立刻跑。
```

Rule:

- Action intensity needs readable path, start/end state, and withheld threat.

## Pattern 8: Fantasy Overload To One Wonder Mechanism

Weak:

```text
森林里有发光门、很多粒子、魔法、奇幻生物。
```

Rewrite direction:

```text
只保留一条蓝光从树根缝隙流出，孩子跟着光线看见门缝。奇观来自光线行为和尺度，而不是堆满元素。
```

Rule:

- One clear wonder is stronger than many simultaneous miracles.
