# Seedance 2.0 Final Prompt Template

Use this exact structure for final prompt-writing tasks.

```markdown
片段名：[片段名称]
时长：[总时长，不超过15秒]
比例：[16:9 / 9:16 / 1:1 / 2.39:1 等]
类型：[真人电影感 / 产品广告 / 青春MV / 动作片段 / CG动画 / 奇幻短片 / 生活方式广告 等]
分镜数量建议：[建议3-8个分镜]
## 核心风格
[整体影像风格、情绪阶段、画面气质、色彩倾向、影像质感、节奏感觉。]
## 场景锁定
[固定场景。明确时间、地点、天气、空间结构、主要环境元素、色彩、光线与必须保留的空间锚点。]
## 角色锁定
### @角色A / 主体A
[外貌、发型、服装、状态、情绪变化、动作倾向。写清楚该角色/主体在片段中的起点与终点。]
### @角色B / 主体B
[如果有第二角色/主体，写清楚外观、位置、状态、动作倾向和与主体A的关系。若没有则删除。]
## 道具锁定
### @道具A
[道具名称、外观、初始位置、使用方式、最终位置。若没有重要道具则删除本模块。]
## 片段核心
[这个片段真正要表达的核心。写清楚情绪任务、视觉重点、人物/产品/空间关系、镜头重点和观众应该感受到什么。]
```
