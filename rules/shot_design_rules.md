# Shot Design Rules

## Every Shot Must Contain

- what appears in frame
- where the subject is
- what action starts
- what action ends
- what the camera does
- what the composition emphasizes
- what the light does
- what the performance shows
- what sound is heard
- whether there is dialogue

## Good Shot Functions

A shot should primarily do one thing:

- establish space
- show object
- reveal character
- trigger movement
- capture reaction
- build relationship
- complete action
- create visual transition
- end with emotional/product emphasis

## Avoid

- generic front-facing composition
- too much story explanation
- abstract emotional labels without visible detail
- unmotivated camera movement
- all shots having the same size and angle
- product/prop appearing without location
- action without start/end state

## Cinematic Upgrade Patterns

Plain:

```text
The character looks outside.
```

Better:

```text
中近景，角色站在窗边偏右三分之一，窗框作为前景切割画面，眼神越过玻璃看向窗外冷色光，手指停在窗帘边缘。
```

Plain:

```text
The product is beautiful.
```

Better:

```text
产品置于画面中央偏下的玻璃台面，瓶身边缘有一圈细冷光，高光沿曲面缓慢滑过，背景保持柔化。
```
