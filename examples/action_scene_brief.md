# Example Brief: Action Scene

User input:

```text
写一个12秒动作片段，一个人从巷子里听到脚步声，回头，躲进阴影里，镜头要紧张。
```

Expected behavior:

- Identify as suspense/action.
- Use clear spatial anchors: alley wall, wet ground, side light, shadow corner.
- Keep action simple: listen, turn, step back, hide.
- Avoid fast running plus complex camera moves in one shot.
- Use sound: footsteps, breath, distant traffic.
