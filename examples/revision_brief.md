# Example Brief: Revision

User input:

```text
第二个镜头太远了，产品看不清，人物动作也太复杂。帮我改稳定一点。
```

Expected behavior:

- Preserve the existing prompt structure.
- Revise only the requested shot unless the surrounding continuity requires small adjustments.
- Move camera closer.
- Increase product proportion.
- Reduce body movement.
- Add clear prop position and hand action.
- Do not mention previous issues in final prompt body.
