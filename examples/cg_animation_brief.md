# Example Brief: CG Animation

User input:

```text
做一个8秒3D动画，小机器人在桌面上推一个比自己大的电池，最后电池亮起来。
```

Expected behavior:

- Identify as CG character/action/product hybrid.
- Lock robot silhouette, tabletop, battery scale, material.
- Split push action and lighting reveal.
- Use clear side view and low tabletop camera.
- Avoid too many mechanical limb actions at once.
